## 🧩 Estrutura
- `index.html` — galeria (página inicial do Pages)
- `docerias/index-01..05.html` — 5 cartões, 1 arquivo autossuficiente cada (HTML+CSS+JS)
- `.github/workflows/deploy.yml` — publicação automática no GitHub Pages

## ✨ Recursos de cada cartão
Hero do produto da semana · cardápio "Mais Vendidos" com botão **Pedir** (WhatsApp pré-preenchido) · prova social · "Como Pedir" · depoimento · CTA final + **barra fixa sticky** · "Salvar Contato" (vCard `.vcf`) · pill "Aberto agora" por horário.

> Dados de contato são placeholders — edite o objeto `PERFIL` no topo do `<script>` de cada cartão.
